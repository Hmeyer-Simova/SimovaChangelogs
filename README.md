# Änderungen in die Extension Publishen

1. Die Extension Version hochziehen(im jeweiligen Ordner die 'vss-extension.json')  
2. Die Task Version hochziehen(im jeweiligen Ordner die 'Task.json') 
3. Extension nach devOps pushen
4. In devOps die 'Create Utilities' build pipeline ausführen für Nav oder Bc
    !!!auf keinen fall das Token oder Variablen ändern, das Token ist für den VSmarketplace und nicht das von devOps!!!
5. Nachschauen ob die Extension Gebaut und Installiert wurde.
